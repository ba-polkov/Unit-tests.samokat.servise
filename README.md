Проект автоматизации UI-тестов сервиса «Самокат»
Тестовый стенд: https://qa-scooter.praktikum-services.ru/

Структура:

- tests - файлы с тестами;
- locators - локаторы, используемые в тестах;
- conftest - фикстуры для запуска тестов;
- data - тестовые данные полей ввода;
- config - ссылки и текстовые значения;
- allure_results - JSON-файлы с результатами выполнения тестов для генерации отчета;
- requirements.txt - внешние зависимости тестов
- README - общая информация по проекту

Тесты:

- test_logo_redirect - проверка переходов на главную страницу сервиса и на главную страницу «Дзена» при клике по логотипу в шапке;
- test_main_page - проверка соответствия текста раскрывающихся ответов в разделе FAQ «Вопросы о важном»
- test_order_page - e2e провекра позитивного сценария оформления заказа с разных точке входа.

Команды для запуска тестов:

- Установка зависимостей - `pip install -r requirements.txt`
- Запуск всех тестов - `pytest -v`
- Запуск всех тестов с формированием файлов для отчета - `pytest tests/ --alluredir=./allure_results`
- Отчет о тестировании (ALlure) - `allure serve allure_results`
